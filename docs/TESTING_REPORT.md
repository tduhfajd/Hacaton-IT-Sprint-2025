**Данные:**
```json

### ШАГ 6: ИИ обрабатывает НОВУЮ тему

**Backend логика:**
1. WebSocket получает сообщение от гражданина
2. Триггерит Celery task: `contextual_response`
3. Извлекает контекст диалога
4. **Ключевые слова:** `["холодной", "воды", "два", "дня"]`

**Поиск по ВСЕЙ базе знаний:**
```
📄 "Водоснабжение и водоотведение" - 80 баллов ⭐
📄 "Теплоснабжение..." - 20 баллов
📄 "Плата за ЖКУ" - 10 баллов

ВЫБРАНА: "Водоснабжение и водоотведение" (80 баллов)
```

**✅ УСПЕХ:** Система нашла статью про **ВОДУ**, а не про отопление!

---

### ШАГ 7: Оператор видит НОВУЮ подсказку

**В чате обновилось:**
```
👤 Гражданин:
