## Где обновлять категории

### 1. База данных
Категории хранятся в таблице `categories`:
```sql
SELECT id, name FROM categories ORDER BY name;
```

Для добавления новой категории:
```sql
INSERT INTO categories (id, name, description, created_at) 
VALUES (gen_random_uuid(), 'Название', 'Описание', NOW());
```

### 2. Telegram бот
Telegram бот **автоматически** загружает категории из БД при вызове команды `/new` или нажатии кнопки "📝 Создать обращение".

**Файл:** `backend/src/services/TelegramBotService.ts`
**Метод:** `showCategorySelection()`

### 3. Веб-интерфейс (форма создания обращения)
**Файл:** `frontend/user/src/App.tsx`
**Строки:** 240-251

Категории жестко заданы в HTML:
```tsx
<select name="subject" ...>
  <option value="">Выберите категорию</option>
  <option value="Благоустройство">Благоустройство</option>
  ...
</select>
```

⚠️ **Важно:** При добавлении новой категории в БД нужно также обновить список в `App.tsx`!
